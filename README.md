# P-P-Maniya-charity

A nonprofit charitable organization serving society through educational, medical, and social welfare activities across India since 2019.

## Features
- Clean and modern responsive UI.
- Interactive hero image carousel.
- Free medical camp and educational program highlights.
- Detailed public trust registration details.
- Validated contact inquiry form.

## Deployment
This project is configured for easy deployment on **Vercel** as a static site.
